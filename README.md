# Info
**Neys Loader - скрипт для платформы Roblox, который поможет вам в плейсах про стрельбу.**
 
*В этом скрипте есть ключ-система через Linkvertise. Ключ действует до изменений скрипта(каждое обновление - новый ключ). Если вам не трудно нажать три раза "Skip ADD", то пожалуйста не крякайте данный скрипт.*

***Связь - @fuckneysonov***

# Functions
**Функции скрипта:**
*AIM: Настраиваемый Aim(Проверка стен, проверка команды, размер FOV, сглаженность при наведении)
CHAMS: Чамсы(Для команды в которой игрок - зеленый, для других - красный)
Misc: Скоро...*
